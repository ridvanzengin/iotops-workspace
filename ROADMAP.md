# Roadmap — moved

This file used to hold the day-by-day working log for Milestone 5 (the
Automation Engine) — decisions made, bugs found, verification narratives.
That content has moved so it lives with the code it describes, and so
this coordination repo doesn't carry a document that outgrows its own
stated scope:

- **Current status, what's open, and the phased plan** (v1 through v1.2
  and beyond) — see
  [IoTOps/docs/development-plan.md](IoTOps/docs/development-plan.md).
- **Dated history of what's shipped** (compressed from this file's old
  DONE sections) — see [IoTOps/CHANGELOG.md](IoTOps/CHANGELOG.md).

If a code comment somewhere points you at "this file's `<topic>` note"
and you don't find it here anymore, it's in one of the two places above —
search CHANGELOG.md for the topic name first; almost everything that used
to live here as a narrative entry is there now, just compressed.

This file is kept at the same path deliberately (rather than deleted
outright) so those existing comments still resolve to something, not a
404 — not because this is still the living document. Update the two docs
above going forward, not this one.
